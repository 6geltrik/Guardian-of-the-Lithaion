# Ο Φύλακας του Ληθαίου

<img src = image/filakas.jpg width = 800>

Την πόλη των Τρικάλων την διαρρέει ένας όμορφος ποταμός, ο Ληθαίος.

Αυτός ο ποταμός σε φυσιολογικές καιρικές συνθήκες ρέει σταθερά και ομαλά.

Όμως σε ακραίες κακοκαιρίες (πχ κακοκαιρία Ntaniel) γίνεται βίαιος και καταστροφικός. 
Επίσης δεν καθαρίζεται σε τακτά χρονικά διαστήματα και η κοίτη του γεμίζει με αυτοφυή φύκια και βλάστηση, με αποτέλεσμα αυτά να συγκρατούν και άλλα σκουπίδια.

Έτσι σκεφτήκαμε να κατασκευάσουμε ένα ρομπότ με τις εξής δυνατότητες.
 Να κινείτε αυτόνομα και να καθαρίζει το ποτάμι κόβοντας την βλάστηση , να κινείται με ηλιακή ενέργεια, να μετράει θερμοκρασία, βροχόπτωση, αγωγιμότητα, βάθος και όλα αυτά τα στοιχεία να τα μεταδίδει μέσω wi fi σε ένα κεντρικό υπολογιστή ώστε να γνωρίζουν οι υπάλληλοι του Δήμου κάθε στιγμή την κατάστασή του και να παίρνουν έγκαιρα μέτρα προστασίας του πληθυσμού, επίσης θα είναι πάντα καθαρός.

# ΚΑΤΑΣΚΕΥΗ

<img src = image/diagrama.jpg width = 800>

Ξεκινήσαμε την κατασκευή με την τοποθέτηση του arduino Mega και το Γυροσκόπιο πάνω στο κόντρα πλακέ

Προγραμματίζουμε το MPU6050 και κάνουμε δοκιμές καλής λειτουργίας. Δηλαδή αν οι γωνίες που παίρνουμε από τον αισθητήρα ανταποκρίνονται πραγματικά. και μετά σε συνδιασμό με τον αισθητήρα υπερήχων που μετρά την απόσταση από την ακτή ελέγχουμε με τα ρελέ την λειτουργία των κινητήρων, ανάλογα με την κατεύθυνση που θέλουμε να κινηθούμε.

Κατόπιν ασχολούμαστε με έναν κάθε φορά αισθητήρα και δοκιμάζουμε την καλή λειτουργία του. Δηλαδή προγραμματίζουμε και ελέγχουμε τον αισθητήρα θερμοκρασίας και υγρασίας, τον αισθητήρα βροχής, τον αισθητήρα αγωγιμότητας, τον αισθητήρα τάσης της μπαταρίας

Δοκιμάζοντας όλα τα παραπάνω να λειτουργούν σωστά, ξεκινάμε τον προγραμματισμό του πομπού wifi ESP8266 και κάνουμε δοκιμές λήψεις δεδομένων από τους αισθητήρες με ένα tablet 4G

Βέβαια για να λειτουργήσουν όλοι οι παραπάνω αισθητήρες χρειάστηκε να αναζητήσουμε πολλούς drivers 

Επόμενο βήμα είναι να σχεδιάζουμε με χαρτόνι τα μηχανικά μέρη που πρέπει να κατασκευάσουμε
 φανταζόμαστε ζωγραφίζουμε και κόβουμε αυτά τα μηχανικά μέρη. Εκτός σχολείου πραγματοποιούμε αυτές τις ιδέες σε πρακτικά αντικείμενα

Δοκιμάζουμε την λειτουργικότητα των μηχανικών μερών και κάνουμε κάποιες βελτιώσεις όπου νομίζουμε ότι θα λειτουργήσουν πιο απότελεσματικά

Τοποθετούμε τα μοτέρ με τους έλικες ( που κατασκευάσαμε με 3D εκτυπωτή ) και κάνουμε πάλι δοκιμές 
Η κίνηση του είναι σύμφωνα με το διάγραμμα

Αφού όλα ανταποκρίνονται κανονικά είμαστε έτοιμοι για την τελική δοκιμή σε πραγματικές συνθήκες στο ποτάμι.
